Gestionnaire de Contacts (POO Java)

Description
Ce projet est une petite application console développée en Java. Elle permet de créer des contacts, de les stocker dans un gestionnaire, et d'effectuer des recherches par nom.

L'objectif principal de ce projet est de mettre en pratique les concepts fondamentaux de la Programmation Orientée Objet (POO).

Concepts clés utilisés
Création de Classes et d'Objets : Modélisation d'une entité Contact et d'un ContactManager.
Constructeurs : Initialisation des états par défaut du gestionnaire (tableau vide, compteur à 0).
Tableaux d'objets : Stockage dynamique (manuel) de plusieurs contacts dans un tableau.
Méthodes d'instance : Ajout (addContact) et recherche (searchContact).
Mot clé this : Pour lever l'ambiguïté entre variables locales et attributs.
Encapsulation de base : Utilisation logique des variables.
Comment lancer le projet ?
Clonez ce dépôt sur votre machine.
Ouvrez le dossier dans IntelliJ IDEA.
Lancez la classe Main.
Le programme affichera le numéro de téléphone du contact recherché dans la console.
